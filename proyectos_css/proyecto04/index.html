Ordenador

Definciones

  • Ordenador: máquina electrónica digital programable que ejecuta una serie de comandos para procesar los datos de entrada, obteniendo convenientemente información que posteriormente se envía a las unidades de salida.

  • Ordenador compatible: capacidad que tienen dos ordenadores de intercambiar información uno con otro simultáneamente.

  • Ordenador personal: tipo de microcomputadora diseñada en principio para ser utilizada por una sola persona. Habitualmente, la sigla PC se refiere a las computadoras IBM PC compatibles. Una computadora personal es generalmente de tamaño medio y es usada por un solo usuario (aunque hay sistemas operativos que permiten varios usuarios simultáneamente, lo que es conocido como multiusuario).

  • Ordenador portátil: Una computadora portátil, computador portátil u ordenador portátil, a veces simplemente conocido como portátil o por los anglicismos laptop y notebook es una computadora personal que se puede mover o transportar con relativa facilidad.
